Jeopardy Project

For our project, we decided that we wanted to work with the Jeopardy dataset . Since we both like gameshows, we wanted to see if there were any interesting findings in this dataset. Our idea was to first try to datamine the dataset to see if there were any interesting trends. Then we wanted to see if we could make a model to classify the prize value of a question based on the question text. Finally, we wanted to make a website to store all of our information.
